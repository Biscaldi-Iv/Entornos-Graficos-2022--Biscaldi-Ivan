¿Qué es CSS y para qué se usa?

Las hojas de estilo en cascada o 'Cascading Style Sheets' (CSS) son documentos que permiten especificar la forma de representar un documento de HTML. De esta forma se puede separar a la información del diseño/formato.

Las hojas de estilo se utilizan para controlar la apariencia de una pagina web, permiten describir la forma que se presentan los elementos de una pagina.

Con CSS se puede especificar el color, tamaño, fuente, espacio entre elementos, recuadros y ubicacion de elementos.

El lenguaje de las Hojas de Estilo está definido en la Especificaciones CSS1, CSS2 y CSS3 del W3C y es un estandard de la web.

CSS utiliza reglas para las declaraciones de estilo, ¿cómo funcionan?

Las hojas de estilo se componene de reglas aplicadas a documentos HTML o XML.
Las reglas se contienen dos partes: selector y declaracion, y a su vez, la declaracion se compone de propiedades y los valores que se le asignan.
El selector especifica que partes del documento o elementos se ven afectados por la declaracion.
La declaracion establece cual es el efecto a producir.

¿Cuáles son las tres formas de dar estilo a un documento?

Se puede declarar el estilo de un documento mediante:

  • el uso del atributo style en cada elemento del documento al que se le quiera dar un estilo
  • la declaracion de estilos en el encabezado( head ) de la pagina
  • la declaracion de estilos en un archivo independiente con extencion .css

¿Cuáles son los distintos tipos de selectores más utilizados?

Ejemplifique cada uno.

Los selectores se pueden clasificar en:

Selectores simples:

Seleccionan los elementos basados en su id, nombre (name), clase (class) o tipo de elemento.
Ej:
div { propiedad: valor; } /* Afecta a todos los elementos div */
#elem { propiedad: valor; } /* Afecta al elemento con id elem */
.clase1 { propiedad: valor; } /* Afecta a los elementos de clase "clase1" */
p.clase2 { propiedad: valor; } /* Afecta a los elementos p de clase "clase2" */
* { propiedad: valor; } /*Afecta a todo el documento */

Selectores combinadores:

Selecciona elementos basados en la relacion con otros elementos
Selectores de descendientes:
Se seleccionan a los elementos especificados que estan contenidos por sus predecesores (el selector se escribe separado por espacio).
Ej:
div p { propiedad: valor; } /* solo afectara a los elementos p contenidos por elementos div */
Selectores de hijos:
Se seleccionan a los elementos especificados que son hijos de un elemento especificado
Ej:
div > p { propiedad: valor; } /* solo afectara a los elementos p que son hijos de elementos div */
Selectores de hermanos adyacentes:
Se seleccionan los elementos que estan inmediatamente luego de determinado elemento
Ej:
div + p { propiedad: valor; } /* afecta a cada elemento p que se encuentre inmediatamente luego de un elemento div */
Selectores generales de hermanos:
Se seleccionan los elementos que estan luego de determinado elemento hermano
Ej:
div ~ p { propiedad: valor; } /* afecta a cada elemento p que se encuentre luego de un elemento div y ademas tengan un padre en comun */

Selectores de pseudoclases:

Las pseudoclases se usan para definir un estado especial de un elemento. Por ejemplo un elemento sobre el cual pasa el mouse o un link que fue o no visitado:
/* link no visitado */
a:link {
color: #FF0000;
}
/* link visitado */
a:visited {
color: #00FF00;
}

Selectores de pseudoelementos:

Un pseudoelemento es una parte de un elemento. Las posibles partes de un elemento pueden ser la primera letra, la primer linea, los marcadores de una lista, la parte seleccionada por un usuario con el mouse, y la parte anterior o posterior de un elemento.
Ej:
p::first-letter { color: #ff0000; font-size: xx-large; } /* modifica el color y tamaño de la primera letra de cada elemento p */

Selectores de atributos

Se pueden seleccionar elementos que tienen atributos especificos. Existen diferentes formas de seleccioonar atributos:
Selector de atributo
Selecciona los elementos que tengan el atributo especifico
Ej:
/*Afecta a todos los links con atributo target*/
a[target] {
background-color: yellow;
}
Selector [atributo="valor"]
Selecciona los elementos con un atributo y su valor especificado
Ej:
/*Afecta los link cuyo target sea igual a "_blank"*/
a[target="_blank"] {
background-color: yellow;
}
Selector [atributo~="valor"]
Selecciona a los elementos con cierto atributo si este en su valor contiene cierta palabra separada por espacios
Ej:
/*Afecta a los elementos cuyo atributo title contenga la palablra 'flower'*/
[title~="flower"] {
border: 5px solid yellow;
}
Selector [atributo|="valor"]
Selecciona elementos con cierto atributo si este tiene el valor especificado o el valor especificado seguido por un guion("-")
La palabra/valor especificados debe estar completa
Ej:
/*Selecciona todos los elementos de clase "top". ademas selecciona por ejemplo elementos donde class="top-text"
pero no seleccionara elementos cuya clase sea por ejemplo "top1"*/
[class|="top"] {
background: yellow;
}
Selector [atributo^="valor"]
Selecciona elementos con cierto atributo si este en tiene un valor que comienza con la secuencia de caracteres especificada
En este caso no es necesario utilizar la palabra completa
Ej:
/*Seleccionara todos los elementos que en la especificacion de clase contengan la cadena "top",
es decir, tanto class="top" como class="top1" seran seleccionados */
[class^="top"] {
background: yellow;
}
Selector [atributo$="valor"]
Selecciona elementos con cierto atributo si este en tiene un valor que termina con la secuencia de caracteres especificada
En este caso no es necesario utilizar la palabra completa
Ej:
/*Selecciona todos los elementos que tengan una clase de terminacion "test".
Por ejemplo tanto class="redtest" como class="bluetest" seran seleccionados*/
[class$="test"] {
background: yellow;
}
Selector [atributo*="valor"]
Selecciona elementos con cierto atributo si este en tiene un valor que contiene la secuencia de caracteres especificada
En este caso no es necesario utilizar la palabra completa
Ej:
/*Selecciona todos los elementos cuya clase en alguna parte tenga la cadena "te"*/
/*Por ejemplo class="tomate" y class="eterno" seran seleccionados*/
[class*="te"] {
background: yellow;
}

¿Qué es una pseudo-clase? Cuáles son las más utilizadas aplicadas a vínculos?

Como se menciono anteriormente, una pseudo-clase es un elemento que tiene un estado especial. Las pseudoclases mas aplicadas a vinculos son:
  • Vinculos visitados -> a:visited
  • Vinculos no visitados -> a:link
  • Vinculos seleccionados -> a:active
  • Vinculo que tiene el mouse encima -> a:hover

¿Qué es la herencia?

La herencia en CSS es el mecanismo mediante el cual determinadas propiedades de un elemento padre se transmiten a sus hijos.

¿En qué consiste el proceso denominado cascada?

Las hojas de estilo son en cascada en cuanto a que el formato/estilo final de cada elemento depende de todas las propiedades que le afectan, es decir, no solo las especificadas para ese elemento, si no tambien la de los elementos que lo contienen. Un elemento puede encontrar en hojas de estilo mas de una declaracion acerca de una determinada propiedad, la declaracion que definira a esa propiedad del elemento siempre sera la ultima.
