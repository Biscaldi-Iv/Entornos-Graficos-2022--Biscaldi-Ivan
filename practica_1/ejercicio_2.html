Segmentos de codigo

    1.
      Esta etiqueta sirve para hacer comentarios. Todo lo escrito entre y es comentario, no sera representado. Este tipo de etiqueta se puede usar en cualquier parte del codigo. 
    2.
      Las etiquetas y sirven para dividir secciones de codigo, dentro del cuerpo del documento. No requiere ningun atributo pero soporta:
      • id: sirve para identificar el elemento
      • class: asigna una clase al elemento, usualmente se usa para apuntar a clases de la hoja de estilos o de JavaScript
      • style: permite definir el estilo del elemento
    3.
      La etiqueta se usa para insertar imagenes en el cuerpo del documento. Crea un espacio para la imagen a la que hara referencia. Esta etiqueta es vacia, solo contiene atributos y no requiere cierre. img requerira 2 atributos:
      • src: especifica la ruta a la imagen
      • alt: especifica un texto alterno que se mostrara en caso de no poder cargar la imagen
      Ademas soporta los siguientes atributos:
      • style
      • width: indica el ancho con el que se muestra la imagen
      • height: indica el largo con el que se muestra la imagen
    4.
      La etiqueta es un elemento que se utiliza en la cabecera de la pagina( ) y proporciona metadatos acerca del documento. Comunmente se utiliza para especificar el formato de caracteres utilizado por la pagina(charset), descripcion de la pagina(description), palabras clave(keywords) y configuracion de la ventana grafica(viewport). Tambien puede usarse para indicar quien es el autor del documento(author) y cuando espira la pagina(http-equiv="expires") o cuando debe refrescarse automaticamente(http-equiv="refresh"). Los atributos que soporta son:
      • charset: indica el formato de caracteres utilizado por la pagina
      • name: se utiliza para señalar que tipo de informacion se encuentra en el atributo content
      • content: contiene informacion/metadatos cuyo tipo fue indicado por name o http-equiv
      • http-equiv: funciona similar a name pero en este caso es para indicar atributos del protocolo http
      La informacion proporcionada en la etiqueta meta no se muesta en la ventana, pero puede indicar al navegador como mostrar la pagina o cuando recargar y servir de guia para motores de busqueda(por las keywords). 
    5.
      La etiqueta se utiliza para colocar hipervinculos en el cuerpo de la pagina. Al hacer click sobre ellos se abren las paginas cuyo enlace fue especificado o tambien se puede mostrar una seccion de la misma pagina cuando el enlace contiene # al comienzo. Dentro de la etiqueta se puede colocar tanto un texto como cualquier otro elemento siendo comun el uso de imagenes. Los atributos de esta etiqueta son:
      • href: indica el enlace de otra pagina cuando este comienza con http:// o https://. Si comienza con # dirigira a un elemento de la misma pagina cuyo id es similar al texto que le sigue al #
      • target: indica si el enlace debe abrirse en la misma pestaña("_self") o en una nueva("_blank"), o abrir el documento en el cuerpo completo de la ventana("_top") o abrir el documento en el marco principal("_parent") 
    6.
      La etiqueta se utiliza para colocar tablas en el cuerpo de la pagina. Una tabla consiste de celdas dentro de filas y columnas.
      Dentro una tabla normalmente se colocan:
      • para agrupar el contenido de los encabezados
      • para agrupar el contenido del cuerpo de la tabla
      • para agrupar el contenido del pie de la tabla
      • especifica un grupo de columnas para formatear
      • especifica las propiedades de una colummna dentro de un grupo de columnas
      • para definir el subtitulo de la tabla
      • para definir el encabezado de una celda
      • para definir una fila en la tabla
      • para definir una celda en alguna fila
